# Техническая документация

## Описание

Скрипт вытягивает xml-схему вида сведений из документов ЕФТТ и формирует json в формате {"название услуги":"xml-схема вида сведений"}.

---

## Установка и запуск

1. Склонируйте репозиторий:
```bash
git clone <URL репозитория>
```
2. Перейдите в папку с проектом:
```bash
cd <директория проекта>
```
3. Установите необходимые пакеты:
```bash
pip install -r requirements.txt
```

## Использование

1. Для корректной работы программы, в папке со скриптом должны быть следующие директории:
- **`docx`**: директория со входными ЕФТТ-документами.
    - **`xml`**: директория с результатом программы.

2. Запустите скрипт.
